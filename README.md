## 🚀 Featured Projects

### 🎸 [Punk Records](https://github.com/Ekaanksh-dev/devops-ai-notifier) — AI-Powered DevOps Commit Notifier

> Watches your GitHub repos and sends AI-generated commit summaries via email — with zero manual setup for new repos.

- 🧠 Uses **LLaMA 3.3 70B via Groq** to analyze every push
- ⚡ New repos connect themselves automatically via a **GitHub App + Flask webhook**
- 🛠️ Ships with `autopwflow` — a custom CLI to manage connected repos

`GitHub Actions` · `Python` · `Groq` · `Flask` · `Render`

---

### 🤖 [AutoDebug](https://github.com/Ekaanksh-dev/AutoDebug) — 5-Agent Autonomous Bug Detection Pipeline

> 5 independent agents that detect, analyse, fix, test, and report bugs — coordinating through Band @mentions. Human always reviews. Nothing auto-merges.

- 🕵️ Detects bugs from CI logs or by running files directly
- 🧠 AI root cause analysis + fix generation via **NVIDIA MiniMax M2.7**
- 📧 Delivers a clean fix report to your inbox — no surprise commits

`Python` · `Band SDK` · `NVIDIA API` · `GitHub API` · `pytest`

---

### 🏥 [Health-agent](https://cdn.botpress.cloud/webchat/v3.6/shareable.html?configUrl=https://files.bpcontent.cloud/2026/05/28/07/20260528072354-XLI8I7ST.json) — RAG-Powered Health Assistant

> A conversational health agent built with Botpress ADK — answers health questions using a curated knowledge base across symptoms, nutrition, fitness, mental health & more.

- 📚 RAG over a structured health knowledge base
- 💬 Covers 15+ health domains — from diabetes to sleep disorders
- 🚀 Live & deployable via `adk deploy`

`TypeScript` · `Botpress ADK` · `RAG`

---

### 🎓 [EduGuide AI Bot](https://github.com/Ekaanksh-dev/Eduguide_org_bot) — Intelligent Academic Assistant

> Full-stack AI academic platform — RAG-based document QA, resume analysis, mock interviews, career profiling & scholarship finder in one monorepo.

- 🔍 **RAG pipeline** with Pinecone vector search + Gemini embeddings
- 📄 PDF resume analysis, AI mock interviews & career gap recommendations
- 🏗️ Spring Boot 3.x backend + Next.js 15 frontend with JWT auth

`Next.js` · `Spring Boot` · `Gemini` · `Pinecone` · `Appwrite`
